{% trans 'First Login' %}

{% blocktrans %} Após efetuar o primeiro login no site Dr. Down, você será redirecionado para a página de editar o perfil, onde será necessário preencher seus dados para que seu cadastro seja completado no sistema, após inseri-los aperte no botão "Atualizar". {% endblocktrans %}

{% blocktrans %} Quando você tiver preenchido os seus dados e atualizado seu perfil, será redirecionado para a página "Meu Perfil", você poderá visualizar seus dados, editar seu perfil, mudar o e-mail cadastrado e excluir sua conta. {% endblocktrans %}

{% trans 'Requests' %}

{% blocktrans %} Apertando em Pedidos na barra lateral você será redirecionado para a página que lista os pedidos de consultas, os pedidos estarão ordenados de acordo com o risco da especialidade requerida pelo paciente e deverão ser marcados de acordo com a prioridade. {% endblocktrans %}

{% blocktrans %} Para marcar uma consulta você deverá clicar no botão "Aceitar" e será redirecionado para a página de marcação de consulta. {% endblocktrans %}

{% blocktrans %} Caso a consulta seja recusada, você deverá escrever uma justificativa para essa consulta ter sido recusada. {% endblocktrans %}

{% trans 'Appointments' %}

{% blocktrans %} Quando você aceitar um pedido de consulta ou selecionar a opção de Nova Consulta na aba de consultas marcadas, será redirecionado para esta página. {% endblocktrans %}

{% blocktrans %} Quando estiver aceitando um pedido, os campos de Especialidade, Profissional de Saúde e Paciente já estarão marcados. O paciente terá escolhido um turno e um dia da semana desejado, você terá que procurar por essa disponibilidade, caso não seja possível, não é obrigatório que seja como desejado pelo paciente. {% endblocktrans %}

{% blocktrans %} Ao clicar em Marcadas na aba de Consultas, você será redirecionado para a lista de consultas marcadas. Você poderá editar ou cancelar uma consulta, caso seja necessário. {% endblocktrans %}

{% trans 'Patients' %}

{% blocktrans %} Ao clicar em "Pacientes", você terá acesso a uma lista de pacientes do Cris Down e ao clicar em um paciente você será redirecionado para o prontuário do mesmo. {% endblocktrans %}

{% trans 'Medical Records' %}

{% blocktrans %} Ao acessar o prontuário de um paciente, poderá visualizar as informações do paciente escolhido e os dados preenchidos pela equipe de saúde do Cris Down: evoluções, dados do pós-natal, exames médicos, prescrições, queixas e curvas de altura, peso, imc e perímetro cefálico. {% endblocktrans %}

{% blocktrans %} Também poderá ir para a Linha de Cuidado ou imprimir/salvar em PDF o seu prontuário. {% endblocktrans %}

{% trans 'Checklist' %}

{% blocktrans %} Ao clicar em "Linha de Cuidado" você terá acesso a procedimentos que o paciente deverá seguir, você poderá também marcar os procedimentos já feitos pelo paciente. São procedimentos que um portador de Síndrome de Down tem que seguir no decorrer de sua vida, alguns são obrigatórios e ourtos são apenas quando necessário pelo paciente. {% endblocktrans %}

{% trans 'Forum' %}

{% blocktrans %} Ao acessar o Fórum, você será redirecionado para a página de Categorias do Fórum, apertando em alguma categoria você irá para os posts da mesma. {% endblocktrans %}

{% blocktrans %} Você terá acesso aos posts da categoria selecionada, também poderá inserir um novo post para a categoria. {% endblocktrans %}

{% blocktrans %} Na página de comentários, você terá acesso aos comentários do post selecionado e também poderá fazer novos comentários no post. {% endblocktrans %}

{% trans 'Events' %}

{% blocktrans %} Em "Eventos", aparecerá a lista de eventos promovidos pelo Cris Down, clicando em "Novo Evento" você será redirecionado para a página de criação de eventos e poderá criar um evento para ser colocado na lista. {% endblocktrans %}

{% blocktrans %} Na página de novo evento é obrigatório o preenchimento das informações a fim de que os usuários estejam bem informados sobre o evento. {% endblocktrans %}

{% blocktrans %} Clicando em um evento você terá acesso as informações do mesmo. {% endblocktrans %}

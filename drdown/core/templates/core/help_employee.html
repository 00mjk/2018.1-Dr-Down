{% trans 'First Login' %}

{% blocktrans %} After you log in to the Dr. Down website, you will be redirected to the profile editing page, where you will need to fill in your details so that your registration is completed in the system, after inserting them, click the "Update" button. {% endblocktrans %}

{% blocktrans %} Once you have filled in your data and updated your profile, you will be redirected to the "My Profile" page, you will be able to view your data, edit your profile, change the registered email and delete your account. {% endblocktrans %}

{% trans 'Notifications' %}

{% blocktrans %} Nas próximas vezes que fizer o login, explicado aqui, você verá a Página de Notificações. Essa página te avisará quando houver novos eventos, novos pedidos de consulta a serem respondidos e te atualizará sobre o fórum. {% endblocktrans %}

{% trans 'Requests' %}

{% blocktrans %} By clicking on Orders in the sidebar you will be redirected to the page that lists the requests for consultations, the orders will be ordered according to the risk of the specialty required by the patient and should be marked according to the priority. {% endblocktrans %}

{% blocktrans %} To mark an appointment you must click on the "Accept" button and you will be redirected to the appointment marking page. {% endblocktrans %}

{% blocktrans %} If the query is denied, you should write a justification for having this query refused. {% endblocktrans %}

{% trans 'Appointments' %}

{% blocktrans %} When you accept a query request or select the New Query option in the marked queries tab, you will be redirected to this page. {% endblocktrans %}

{% blocktrans %} When you are redirected to create a query through an order, the Specialty, Health Professional and Patient fields are already marked. The patient will have chosen a shift and a desired day of the week, you will have to look for this availability, if it is not possible, it is not mandatory that it be as desired by the patient. {% endblocktrans %}

{% blocktrans %} When you click Marked on the Queries tab, you will be redirected to the list of marked queries. You can edit or cancel a query, if necessary. {% endblocktrans %}

{% trans 'Patients' %}

{% blocktrans %} By clicking on "Patients" you will have access to a list of Cris Down patients and clicking on a patient you will be redirected to the patient's chart. {% endblocktrans %}

{% trans 'Medical Records' %}

{% blocktrans %} When you access a patient's chart, you can view the information of the chosen patient and the data filled by the Cris Down health team: evolutions, postnatal data, medical exams, prescriptions, complaints and curves of height, weight, cephalic perimeter. {% endblocktrans %}

{% blocktrans %} You can also go to the Care Line or print / save your medical records in PDF. {% endblocktrans %}

{% trans 'Checklist' %}

{% blocktrans %} By clicking on "Care Line" you will have access to procedures that the patient should follow. These are procedures that a Down Syndrome sufferer has to follow throughout his or her life, some are compulsory, and weights are only when needed by the patient. {% endblocktrans %}

{% trans 'Forum' %}

{% blocktrans %} By accessing the Forum, you will be redirected to the Forum Categories page, clicking on any category you will go to the forum posts. {% endblocktrans %}

{% blocktrans %} You will have access to the posts of the selected category, you can also insert a new post for the category. {% endblocktrans %}

{% blocktrans %} In the comments page, you will have access to the comments of the selected post and you will also be able to make new comments in the post. {% endblocktrans %}

{% trans 'Events' %}

{% blocktrans %} In "Events", the list of events promoted by Cris Down will appear, clicking on "New Event" will be redirected to the event creation page and you can create an event to be placed on the list. {% endblocktrans %}

{% blocktrans %} In the page of new event it is mandatory to fill in the information in order that the users are well informed about the event. {% endblocktrans %}

{% blocktrans %} By clicking on an event you will have access to the event information. {% endblocktrans %}

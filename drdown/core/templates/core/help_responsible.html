{% trans 'First Login' %}

{% blocktrans %} After you log in to the Dr. Down website, you will be redirected to the profile editing page, where you will need to fill in your details so that your registration is completed in the system, after inserting them, click the "Update" button. {% endblocktrans %}

{% blocktrans %} Once you have filled in your data and updated your profile, you will be redirected to the "My Profile" page, you will be able to view your data, edit your profile, change the registered email and delete your account. {% endblocktrans %}

{% trans 'Requests' %}

{% blocktrans %} By clicking on Orders in the sidebar you will be redirected to the page that lists the requests for inquiries made by you for whom you are responsible. {% endblocktrans %}

{% blocktrans %} In this page you can create a new request for a query, edit an order or delete an order. {% endblocktrans %}

{% trans 'Marked Appointments' %}

{% blocktrans %} By clicking on "Marked" in the Queries tab, you will be redirected to the list of queries that have been marked for you by a Cris Down employee, where you can view the specialty, date and time. {% endblocktrans %}

{% trans 'Vaccine Schedule' %}

{% blocktrans %} Clicking on the vaccine calendar will show a table with two tabs that indicate the vaccines by age, the first flap is from birth to six years of age and the second flap is nine years old and up. {% endblocktrans %}

{% trans 'Vaccination Sites' %}

{% blocktrans %} After accessing "Vaccination Sites" in the sidebar, you will be redirected to a list of cities in the Federal District, where each city has vaccination clinics. {% endblocktrans %}

{% trans 'Forum' %}

{% blocktrans %} By accessing the Forum, you will be redirected to the Forum Categories page, clicking on any category you will go to the forum posts. {% endblocktrans %}

{% blocktrans %} You will have access to the posts of the selected category, you can also insert a new post for the category. {% endblocktrans %}

{% blocktrans %} In the comments page, you will have access to the comments of the selected post and you will also be able to make new comments in the post. {% endblocktrans %}

{% trans 'Events' %}

{% blocktrans %} In "Events", you will have access to the events promoted by Cris Down, clicking on the event name will appear the information. {% endblocktrans %}

{% trans 'First Login' %}

{% blocktrans %} Após efetuar o primeiro login no site Dr. Down, você será redirecionado para a página de editar o perfil, onde será necessário preencher seus dados para que seu cadastro seja completado no sistema, após inseri-los aperte no botão "Atualizar". {% endblocktrans %}

{% blocktrans %} Quando você tiver preenchido os seus dados e atualizado seu perfil, será redirecionado para a página "Meu Perfil", você poderá visualizar seus dados, editar seu perfil, mudar o e-mail cadastrado e excluir sua conta. {% endblocktrans %}

{% trans 'Requests' %}

{% blocktrans %} Apertando em Pedidos na barra lateral você será redirecionado para a página que lista os pedidos de consultas feitos por você para quem você se responsabiliza. {% endblocktrans %}

{% blocktrans %} Nessa página você poderá criar um novo pedido para uma consulta, editar um pedido ou excluir um pedido. {% endblocktrans %}

{% trans 'Marked Appointments' %}

{% blocktrans %} Ao clicar em "Marcadas" na aba de Consultas, você será redirecionado para a lista de consultas que foram marcadas para você por algum funcionário do Cris Down, onde poderá visualizar a especialidade, data e horário. {% endblocktrans %}

{% trans 'Vaccine Schedule' %}

{% blocktrans %} Ao clicar em calendário de vacinas, irá aparecer uma tabela com duas abas que indicam as vacinas por idade, a primeira aba é do nascimento até os seis anos de idade e a segunda aba é dos nove anos de idade em diante. {% endblocktrans %}

{% trans 'Vaccination Sites' %}

{% blocktrans %} Após acessar "Locais de Vacinação" na barra lateral, você será redirecionado para uma lista de cidades do Distrito Federal, em que cada cidade dispõe de postos de saúde para vacinação. {% endblocktrans %}

{% trans 'Forum' %}

{% blocktrans %} Ao acessar o Fórum, você será redirecionado para a página de Categorias do Fórum, apertando em alguma categoria você irá para os posts da mesma. {% endblocktrans %}

{% blocktrans %} Você terá acesso aos posts da categoria selecionada, também poderá inserir um novo post para a categoria. {% endblocktrans %}

{% blocktrans %} Na página de comentários, você terá acesso aos comentários do post selecionado e também poderá fazer novos comentários no post. {% endblocktrans %}

{% trans 'Events' %}

{% blocktrans %} Em "Eventos", você terá acesso aos eventos promovidos pelo Cris Down, clicando no nome do evento aparecerá as informações. {% endblocktrans %}
